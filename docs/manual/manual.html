1. Introducción

Jaguata es una plataforma web que conecta dueños de mascotas con paseadores verificados, permitiendo reservar paseos, gestionar solicitudes, registrar pagos, generar comprobantes, recibir notificaciones y administrar perfiles según el rol.

2. Alcance y roles

El sistema se divide en 3 roles con permisos y menús distintos.

Roles

  • Administrador: gestiona usuarios, pagos, auditoría, reportes y configuraciones del sistema.
  • Dueño: registra mascotas, busca paseadores, solicita paseos, realiza pagos y califica.
  • Paseador: gestiona solicitudes, disponibilidad, paseos, cobros/remuneraciones y suscripción.
Acceso por rol: al iniciar sesión, cada usuario ingresa a su panel correspondiente y solo ve los módulos habilitados para su perfil.

3. Requisitos del sistema

Requisitos mínimos para usar Jaguata correctamente.

Para usuarios finales

  • Navegador actualizado: Chrome, Edge o Firefox.
  • Conexión a internet estable.
  • Permitir ubicación (recomendado) para seleccionar el punto de retiro y detectar ciudad.

4. Pantallas públicas, acceso y registro

Esta sección describe el recorrido completo: primero el sitio público (sin iniciar sesión) y, luego, el ingreso al sistema (login/registro/recuperación de contraseña).

4.1 Pantallas públicas (antes de iniciar sesión)

Pantalla principal del sistema (sitio público): el acceso inicial se realiza desde public/sobre_nosotros.php. Desde esta pantalla se presenta Jaguata y se ofrecen accesos a Registro, Login y otras secciones.
Acceso en localhost: http://localhost/jaguata/public/sobre_nosotros.php
  • Home / Landing: explica qué es Jaguata y muestra botones de acceso (registro/login).
  • Contacto: permite enviar un mensaje (nombre, email y contenido).
  • Bases y condiciones: muestra términos de uso y responsabilidades.
Tip: si el usuario ya está logueado, el sistema puede mostrar acceso directo al Dashboard según su rol.

4.2 Iniciar sesión (Login)

  1. Ingresá a la pantalla Login.
  2. Escribí tu email y contraseña.
  3. Presioná Ingresar.
  4. Si la cuenta está habilitada, el sistema redirige a tu Dashboard según rol.
Estado de cuenta: si tu usuario está pendiente, rechazado, suspendido o inactivo, el sistema puede bloquear el acceso total o parcial hasta que el administrador lo habilite.

4.3 Registro de usuario

  1. Entrá a Registro.
  2. Completá tus datos personales.
  3. Elegí el rol correspondiente: Dueño o Paseador.
  4. Confirmá el registro. La cuenta queda en estado pendiente hasta revisión/aprobación.

4.4 Recuperación de contraseña

  • En Login, seleccioná “Recuperar contraseña”.
  • Ingresá tu email y seguí las instrucciones del sistema.
  • Definí una nueva contraseña segura (recomendado: mínimo 8 caracteres con letras y números).
Recomendación: no compartas tu contraseña y cerrá sesión si usás equipos públicos.

4.5 Contacto y bases

  • Contacto: completá nombre, email y mensaje para enviar una consulta o problema.
  • Bases y condiciones: revisá los términos de uso antes de operar en la plataforma.
Tip para soporte: al reportar un problema, indicá la pantalla/módulo y adjuntá una captura si es posible.

4.6 Capturas de pantallas (Público y Acceso)

Guardá tus capturas en docs/manual/img/ y usá los nombres sugeridos.

5. Navegación general

Estructura común de los paneles: sidebar + contenido + módulos.

  • Sidebar (menú lateral): muestra los módulos disponibles según el rol.
  • Contenido principal: tarjetas, tablas y formularios (altas, bajas, edición).
  • Notificaciones: lista de avisos importantes (eventos del sistema).
  • Cerrar sesión: finaliza la sesión y vuelve al login.

Capturas reales de los paneles principales del sistema.

7. Rol Administrador

El Administrador es responsable de la operación del sistema: gestión de usuarios, control de pagos, auditoría, reportes y parámetros globales.

Módulos disponibles (menú Admin): Dashboard, Usuarios, Reportes/Estadísticas, Notificaciones, Pagos (efectuados/pendientes), Auditoría, Configuraciones Generales, Roles y Permisos.

7.1 Usuarios

  1. Ingresá a Usuarios para visualizar la lista de cuentas registradas.
  2. Utilizá filtros o búsqueda para localizar un usuario específico.
  3. Accedé a Editar para modificar datos básicos y el estado del usuario.
Estados del usuario: pendiente, aprobado, rechazado, suspendido o inactivo. El estado determina el acceso del usuario a los módulos del sistema.

7.2 Pagos

  • Pagos pendientes: revisión de transacciones aún no confirmadas.
  • Pagos efectuados: historial de pagos confirmados en el sistema.
  • Detalle de pago: visualización de información completa del pago y comprobantes.

7.3 Auditoría

El módulo de auditoría registra todas las acciones relevantes realizadas en el sistema para garantizar trazabilidad y control.

  • Consulta de eventos por fecha, módulo o tipo de acción.
  • Registro de acciones como crear, editar, eliminar, login y cambios de configuración.

7.4 Configuraciones

  • Configuraciones generales: parámetros globales del sistema.
  • Roles y permisos: definición de accesos a módulos según rol.

7.5 Reportes y estadísticas

  • Visualización de métricas generales del sistema.
  • Estadísticas de usuarios, paseos, pagos y actividad.

8. Rol Dueño

El Dueño gestiona sus mascotas, busca paseadores, solicita paseos, realiza pagos, visualiza historial y califica al paseador.

Módulos disponibles (menú Dueño): Dashboard, Reserva (Solicitar Paseo / Mis Paseos), Seguimiento, Buscar Paseadores, Pagos (Gastos Totales / Pagos / Comprobantes), Notificaciones, Mascotas (Mis Mascotas / Agregar / Editar), Perfil (Mi Perfil / Editar), Mis Puntos.

8.1 Gestión de Mascotas

  1. Ingresá a Mis Mascotas para ver el listado.
  2. Seleccioná Nueva Mascota para registrar los datos y una foto.
  3. Desde cada mascota podés Ver Perfil o Editar.
Requisito: para solicitar un paseo necesitás al menos 1 mascota registrada.

8.2 Buscar paseadores

  • Ingresá a Buscar Paseadores.
  • Revisá el perfil del paseador y su reputación.
  • Si corresponde, desde allí podés ir a Solicitar Paseo.

8.3 Solicitar Paseo (Reserva)

La solicitud guía al dueño por pasos: selección de mascota(s), ubicación en el mapa, fecha/hora y confirmación del total.

  1. Seleccioná Mascota 1 (obligatorio).
  2. Opcional: seleccioná Mascota 2 (máximo 2, sin repetir).
  3. Definí la ubicación mediante el mapa (pin) o ubicación del navegador.
  4. Elegí fecha/hora, duración y confirmá el total.
  5. Presioná Solicitar Paseo.
Reglas automáticas:
  • Se permite seleccionar hasta 2 mascotas.
  • No se puede repetir la misma mascota en Mascota 1 y Mascota 2.
  • Si son 2 mascotas, el sistema aplica 30% de descuento automáticamente.

8.4 Mis Paseos y seguimiento

  • Mis Paseos: lista general de solicitudes y paseos.
  • Pendientes: solicitudes por confirmar o en proceso.
  • Completos: paseos finalizados (habilita calificación y comprobantes).
  • Cancelados: historial de cancelaciones.

8.5 Pagos y comprobantes

  • Gastos Totales: resumen de gastos.
  • Pagos: historial de pagos realizados.
  • Comprobantes: vista/descarga de comprobantes asociados.

8.6 Notificaciones

  • Visualización de eventos del sistema: solicitudes, cambios de estado y confirmaciones.

8.7 Calificar Paseador

  • Luego de un paseo completado, ingresá a Calificar Paseador.
  • Seleccioná estrellas (1 a 5) y dejá un comentario (opcional).

8.8 Perfil y puntos

  • Mi Perfil / Editar Perfil: actualización de datos personales.
  • Mis Puntos: consulta del saldo acumulado.

9. Rol Paseador

El Paseador recibe solicitudes, confirma o rechaza, gestiona la ejecución del paseo, administra su disponibilidad, visualiza remuneraciones y gestiona su perfil/suscripción (si aplica).

Módulos disponibles (menú Paseador): Dashboard, Estadísticas, Solicitudes, Mis Paseos, Pagos (remuneraciones), Notificaciones, Perfil (Mi Perfil / Editar Perfil), Disponibilidad, Configuraciones.

9.1 Solicitudes (entrantes)

  1. Ingresá a Solicitudes para ver pedidos pendientes.
  2. Revisá detalle: dueño, mascota(s), ubicación, fecha/hora y duración.
  3. Elegí una acción: Aceptar o Rechazar según corresponda.
Bloqueo por estado/suscripción: si la cuenta no está aprobada o si la suscripción está vencida (si aplica), algunas acciones pueden quedar restringidas.

9.2 Mis Paseos (gestión operativa)

  • Visualización de paseos activos, pendientes y completados.
  • Acceso a Detalle del paseo.
  • Acciones típicas: iniciar, completar o cancelar (según reglas del sistema).

9.3 Pagos y remuneraciones

  • Resumen de ganancias por período.
  • Detalle por paseo completado (si está habilitado).

9.4 Disponibilidad

  1. Ingresá a Disponibilidad.
  2. Marcá días activos y horarios (inicio/fin).
  3. Guardá cambios para que el sistema te muestre como disponible.

9.5 Perfil

  • Mi Perfil: consulta de datos y estado.
  • Editar Perfil: actualización de información personal y datos de pago (si aplica).

9.6 Notificaciones

  • Eventos: nuevas solicitudes, cambios de estado, confirmaciones y avisos del sistema.

9.7 Suscripción (si aplica)

  • Estado: activa / vencida / pendiente.
  • Cuando está activa, puede habilitar funcionalidades según configuración del sistema.

10. Preguntas frecuentes

Respuestas rápidas a problemas comunes.

No puedo solicitar un paseo

  • Verificá que tu cuenta esté aprobada.
  • Confirmá que tenés al menos 1 mascota registrada.
  • Probá seleccionar ubicación en el mapa para cargar paseadores cercanos.

No me aparecen paseadores disponibles

  • Definí una ubicación válida (pin en el mapa).
  • Probá mover el pin o permitir ubicación del navegador.
  • Verificá fecha/hora y duración según disponibilidad.

¿Puedo elegir 2 mascotas?

Sí. El sistema permite hasta 2 mascotas, no repetidas. Si son 2, aplica 30% de descuento.

No puedo ingresar al panel

  • Revisá credenciales (email/contraseña).
  • Si está pendiente/suspendida/inactiva, contactá al administrador.
  • Si olvidaste la contraseña, usá recuperación.

11. Soporte y buenas prácticas

Recomendaciones para un uso seguro y ordenado del sistema.

Buenas prácticas

  • Usá contraseñas seguras y no las compartas.
  • Verificá la información del paseo antes de confirmar (fecha, ubicación, mascotas).
  • Subí comprobantes claros (si se solicita) para evitar rechazos.
  • Calificá con responsabilidad: las calificaciones ayudan a la reputación del sistema.

Contacto soporte

  • Email: soporte@jaguata.com (o el correo definido en configuración del sistema)
  • Horario: 08:00 a 18:00 (referencial)
